# KLAY API PLATFORM

## 목적
KLAYTN 플랫폼에 접근하기 위한 API 서비스

## 기능
- EN 접속 기능
- EN 접근용 SVC key / password 발급 기능
- 개인 정보는 미관리
- EN RPC Proxy
- klaytn public api 사용 API 제공
- 플랫폼만 접근 개인의 접근 허용 안함.
- 개인의 정보는 계좌 정보만 관리

## 사용주의
- 현재 klaytn EN 서버로는 노틸러스의 서버를 사용하고 있음.
- API 기능을 사용하기 때문에 

## 실행법
~~~
npm install
node app.js
~~~
